# 💻 CodeAlpha C++ Internship Tasks

This repository contains my completed projects for the **CodeAlpha C++ Internship Program**.  
The tasks focus on core C++ concepts including basic programming, file handling, and object-oriented programming.

## ✅ Completed Tasks

- Calculates CGPA based on user input
- Uses loops and basic arithmetic
- Handles multiple courses

### 🔐 Login & Registration System
- Allows user registration and login
- Stores data in a file (`users.txt`)
- Demonstrates file handling in C++

### 🏦 Simple Banking System
- Simulates basic banking operations
- Supports deposit, withdrawal, and balance check
- Built using Object-Oriented Programming (OOP)

# 🏦 Simple Banking System (C++)

This program simulates basic banking operations using Object-Oriented Programming in C++.

## 📌 Features
- Create a bank account
- Deposit money
- Withdraw money with balance check
- View current account balance

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (Classes & Methods)

## ▶️ How to Run
```g++ SimpleBanking.cpp -o bank
./bank
```

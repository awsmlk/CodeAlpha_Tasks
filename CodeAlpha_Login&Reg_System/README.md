# 🔐 Login & Registration System (C++)

This program provides a simple login and registration system using file handling in C++.

## 📌 Features
- Register a new user with username and password
- Store user credentials in a file (users.txt)
- Login using stored credentials
- Displays success or error messages

## 🛠️ Technologies Used
- C++
- File Handling (fstream)

## ▶️ How to Run
g++ LoginRegSystem.cpp -o login
./login

## 📂 File Used
- users.txt → stores usernames and passwords

## 📷 Sample Run

2. Login

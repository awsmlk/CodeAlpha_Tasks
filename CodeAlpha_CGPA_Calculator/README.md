# 🎓 CGPA Calculator (C++)

This program calculates the CGPA of a student based on user input of grade points and credit hours.

## 📌 Features
- Takes total number of courses as input
- Accepts grade points and credit hours for each course
- Calculates total credits and total grade points
- Displays final CGPA

## 🛠️ Technologies Used
- C++

## ▶️ How to Run
g++ CGPA_Calculator.cpp -o cgpa
./cgpa
